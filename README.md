# nagarjuna_nallimilli

**Problem1.java – Simple Calculator**

A basic calculator program that performs:

Addition

Subtraction

Multiplication

Division

Features

Accepts two numbers and an operation symbol.

Handles invalid operations.

Prevents division by zero.

**Problem2.java – Generate N Odd Numbers**

Generates the first N odd numbers.

Example

Input: 5
Output: 1,3,5,7,9

Features

Uses loop to generate odd numbers.

Returns output as a comma-separated string.

Returns empty output for invalid inputs.

**Problem3.java – Generate Odd Numbers Up To N**

Generates all odd numbers up to a given number N.

Example

Input: 10
Output: 1,3,5,7,9

Features

Automatically adjusts if N is even.

Returns comma-separated odd numbers until N.

Handles invalid and zero inputs.

**Problem4.java – Count Multiples from Array**

Counts how many numbers in a given array are divisible by 1 to 9.

Example Input Array
1, 2, 8, 9, 12, 46, 76, 82, 15, 20, 30

Example Output
{1=11, 2=8, 3=4, 4=3, 5=3, 6=2, 7=0, 8=1, 9=1}

Features

Uses nested loops to check divisibility.

Returns a map containing divisors (1–9) and their counts.

Scanner version included for user input.

****How to Run the Programs**

Each problem is a standalone Java class. You can run them individually.

 Option 1: Run Using VS Code / IntelliJ

Open the project folder.

Go to Tasks/ProblemX.java.

Click the Run  button.

Provide input if asked.

 Option 2: Run Using Command Line
Step 1: Go to your project directory
cd <your-project-folder>

Step 2: Compile the program

Example (Problem1):

javac Tasks/Problem1.java

Step 3: Run the program
java Tasks.Problem1

Run Commands for All Problems
